Akaito

A light theme for Omarchy inspired by ink-brush posters, red-sun brutalism, and printed paper textures.

Warm whites, deep ink blacks, and restrained blood-red accents. Opinionated, high-contrast, and meant to feel printed, not polished.

Aesthetic

Paper-white background

Ink-black foreground

Muted red accents

Poster / zine energy

Comfortable for long sessions

Install
mkdir -p ~/.config/omarchy/themes/akaito
cp -r akaito ~/.config/omarchy/themes/
./make-colors.sh ~/.config/omarchy/themes/akaito

Apply via Omarchy.

Notes

Light, warm, and intentionally non-neutral

Generated colors for consistency

Best paired with strong wallpapers
